The Product
What EigenView Is
A daily intelligence brief delivered as a live dashboard. The engine runs 5 signal layers across a curated universe of instruments. The output is a ranked, scored, labeled list — not data to interpret. You open EigenView, you see what to look at today, why, and how to trade it.
Layer 1 — Macro Gate
Regime Filter
Before any instrument is scored, the macro regime is evaluated: SPX GEX (SpotGamma), VIX M1/M2 term structure (VIXCentral), DIX dark pool index (SqueezeMetrics), SPX breadth. Score 0–4. GREEN = screener runs. RED = only short setups or cash signal shown. This alone prevents Sep–Mar trap.
Layer 2 — Instrument Scoring
5-Signal EigenView Engine
Each instrument scored 0–10 across: GEX regime (stock-level), dark pool accumulation clusters, IV Rank vs 52-week range, unusual options flow ($1M+ sweeps), post-OpEx gamma clearing window. Score 6+ = enters the curated list. Everything below is filtered out.
Layer 3 — Output
The Curated List
Max 10 instruments per day. Each card shows: ticker, score, setup type, timeframe, entry zone, stop logic, conviction drivers. Not a watchlist. Not a feed. A ranked, opinionated, ready-to-act lineup — stocks AND futures, every session.
Output Type 1
REVERSAL
GEX flip cross + capitulation signals converging. Swing 3–10 days. Entry zone, stop below dark pool cluster. These fire ~8–12x/year at index level.
Output Type 2
CONTINUATION
Breadth healthy, VIX contango restoring, net premium bullish. Re-entry on pullbacks in confirmed trending instruments. Most common output type.
Output Type 3
POSITION
Full capitulation checklist fired (82% T+60 win rate). 2–6 week horizon. LEAPS or stock. Sized for holds through noise. Rare — maybe 3–4x/year.
Output Type 4
INVEST
Valuation reset + capitulation + fundamental support. 3–12 month horizon. Specifically for non-traders who want structural entry points, not day trades.
UI / UX
What It Looks Like
