BUGS FOUND AND FIXED DURING IMPLEMENTATION
=============================================================================

Bug 1 — failed_breakout logic: argrelextrema always detected the spike bar as
  the "last swing high", making exceeded_recently always False.
  Fix: filter swing_high_idx to prior_swing_idx = swing_high_idx[swing_high_idx < 56]
  so only established (non-recent) swing highs are used as the reference level.

Bug 2 — failed_breakdown outer elif consuming elif chain: the outer elif
  condition for failed_breakdown was too broad ('low' in df.columns and len >= 6
  and weekly != bearish_strong). When inner conditions (recent_dip, recovered)
  failed, the chain stopped — breakout never ran. Also: no guard against
  close_now > recent_high (a clean breakout should not match failed_breakdown).
  Fix: merged inner conditions into the outer elif, added close_now <= recent_high guard.
